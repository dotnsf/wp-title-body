# wp-title-body

## Overview

Sample app to retrieve `title` and `body` from WordPress DB.


## Environment values

- `DATABASE_URL` : URL connection string for MySQL

- `DB_LIMIT` : "limit" for selecting SQL

- `DB_OFFSET` : "offset" for selecting SQL
